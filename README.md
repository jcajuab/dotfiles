# dotfiles

My development tool configurations, structured like a standard Linux home directory.

## Getting Started

Clone this repository:

```sh
git clone https://github.com/jcajuab/dotfiles.git ~/.dotfiles
```

## Usage

Link a config:

```sh
# Example: linking `foot` configuration
ln -sf ~/.dotfiles/.config/foot ~/.config/foot
```

- `ln -s`: Create a symbolic link
- `-f`: Overwrite existing files or links
